# learn-blender
resources for learning bleander

## Navigation
* ```sift+f``` first person

## Edit Mode
* ```alt+r``` while in edge mode will create and edge ring
* ```i``` inline extrude and scale a face
* ```e``` extrude
* ```f``` create a new face

### Selection
* ```c``` circle select
* ```b``` box select
* ```shift+alt``` edge ring select
